# zilev-devops/vs-code-extension-git-pack

- NodeJS 14.15.4
- Yeoman 4.3.1
- vsce 2.15.0

## Steps
- npm install -g yo generate-code
- npm install -g vsce
- yo code
- Modify package.json
- vsce package
